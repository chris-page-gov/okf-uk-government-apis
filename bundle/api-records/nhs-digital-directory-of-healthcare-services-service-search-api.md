# Directory of Healthcare Services (Service Search) API

Use this API to find information about organisations that provide NHS healthcare services, NHS organisation types, and coronavirus (COVID-19) walk-in sites.You can:- retrieve a list of organisation types, for example, GP practices or hospitals- search for NHS organisations that provide healthcare services listed in NHS services near you, for example, Leeds General Infirmary- search for coronavirus (COVID-19) walk-in sitesThis API was previously known as the Service Search API.
