# GP2GP - HL7 V3

Use this integration to transfer patients' electronic health records between old and new practices when they change GPs.You can:- include large records and those with many attachments- reduce paper printing when patients leave a practice- integrate (file) the electronic health record for returning patients- log issues easily with easy-to-understand and more informative error messages- monitor in real time the processes to track issues and performanceFor more details, see GP2GP.
